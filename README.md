# 🏙️ 3D Animated Real Estate Platform (MERN Stack)

A premium full-stack real estate web application featuring immersive 3D animations, cinematic scrolling, and modern UI/UX. Built using the MERN stack with a strong focus on performance, interactivity, and conversion-driven design.

---

## 🚀 Features

### 🎬 Frontend (Client)
- ⚛️ Built with React (Vite)
- 🌐 3D experiences using Three.js
- 🎞️ Smooth animations with GSAP + ScrollTrigger
- 🧠 Micro-interactions via Framer Motion
- 🖱️ Parallax + smooth scrolling (Lenis / Locomotive Scroll)
- 🎨 Dark luxury UI (custom CSS / SCSS)
- 📱 Fully responsive design

### ⚙️ Backend (Server)
- 🟢 Node.js + Express.js REST API
- 🍃 MongoDB for data storage
- 📦 Property management (CRUD APIs)
- 📩 Lead capture (contact form)
- 🔐 (Optional) Admin dashboard for property control

---

## 🧩 Tech Stack

### Frontend
